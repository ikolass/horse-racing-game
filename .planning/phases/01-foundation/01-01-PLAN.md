```js
export function shuffle(array) {
  const result = [...array];
  for (let i = result.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [result[i], result[j]] = [result[j], result[i]];
  }
  return result;
}
```

Create `src/utils/raceEngine.js` implementing weighted-random selection per D-03 and D-04.

The function `computeFinishOrder(horses)` receives an array of horse objects (each with a `condition` field) and returns an array of indices representing finish positions (index 0 of result = 1st place).

Algorithm — weighted selection without replacement (per D-03 linear weighting, D-04 no floor):
1. Build a pool of `{ index, weight: horse.condition }` for each horse
2. Loop until pool is empty:
   a. Compute totalWeight = sum of all remaining weights
   b. Pick a random number in [0, totalWeight)
   c. Walk the pool accumulating weights until the random number is exceeded — that horse is selected
   d. Push the selected horse's index to the result array
   e. Remove the selected horse from the pool
3. Return the result array

```js
export function computeFinishOrder(horses) {
  if (!horses || horses.length === 0) return [];

  const pool = horses.map((horse, index) => ({
    index,
    weight: horse.condition,
  }));

  const result = [];

  while (pool.length > 0) {
    const totalWeight = pool.reduce((sum, h) => sum + h.weight, 0);
    let random = Math.random() * totalWeight;

    let selectedIdx = 0;
    for (let i = 0; i < pool.length; i++) {
      random -= pool[i].weight;
      if (random <= 0) {
        selectedIdx = i;
        break;
      }
    }

    result.push(pool[selectedIdx].index);
    pool.splice(selectedIdx, 1);
  }

  return result;
}
```

Weight = raw condition score. No minimum base weight added (per D-04). A horse with condition 80 is exactly twice as likely to be picked next as one with condition 40 (per D-03).
