  <action>
    **GameControls.vue** — Vue 3 SFC using `<script setup>`:

    Import `useStore` from `vuex` and `computed` from `vue`.

    Setup:
    ```javascript
    const store = useStore()
    const gamePhase = computed(() => store.getters['race/gamePhase'])

    const generateDisabled = computed(() => gamePhase.value === 'RACING')
    const startDisabled = computed(() =>
      gamePhase.value === 'IDLE' || gamePhase.value === 'RACING' || gamePhase.value === 'DONE'
    )

    function handleGenerate() {
      store.dispatch('schedule/generateSchedule')
      store.dispatch('race/transitionTo', 'SCHEDULED')
    }

    function handleStart() {
      store.dispatch('race/startRace')
    }
    ```

    CRITICAL: `handleGenerate` dispatches TWO actions — `schedule/generateSchedule` to create the rounds data AND `race/transitionTo('SCHEDULED')` to advance the FSM. The schedule module's `generateSchedule` action does NOT transition gamePhase on its own.

    State/Phase gating matrix (from UI-SPEC):
    | gamePhase      | Generate | Start    |
    |----------------|----------|----------|
    | IDLE           | Enabled  | Disabled |
    | SCHEDULED      | Enabled  | Enabled  |
    | RACING         | Disabled | Disabled |
    | ROUND_COMPLETE | Enabled  | Enabled  |
    | DONE           | Enabled  | Disabled |

    The `startDisabled` computed covers IDLE, RACING, and DONE. Generate is only disabled during RACING. This matches the matrix exactly.

    Template:
    ```html
    <div class="game-controls">
      <button
        class="btn btn-generate"
        data-testid="btn-generate"
        :disabled="generateDisabled"
        @click="handleGenerate"
      >
        Generate Schedule
      </button>
      <button
        class="btn btn-start"
        data-testid="btn-start"
        :disabled="startDisabled"
        @click="handleStart"
      >
        Start Race
      </button>
    </div>
    ```

    Scoped styles:
    ```css
    .game-controls {
      display: flex;
      flex-direction: column;
      gap: var(--space-sm);
      padding: var(--space-lg) var(--space-md);
    }

    .btn {
      width: 100%;
      min-height: 36px;
      border: none;
      border-radius: 6px;
      font-size: var(--font-heading);
      font-weight: var(--font-weight-semibold);
      cursor: pointer;

    }

    .btn:disabled {
      background: var(--color-btn-disabled-bg);
      color: var(--color-btn-disabled-text);
      cursor: not-allowed;
      opacity: 0.6;
    }
    ```
