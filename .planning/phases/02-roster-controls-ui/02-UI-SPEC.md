| Property | Value |
|----------|-------|

## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline icon-to-text spacing |
| sm | 8px | Roster row internal padding (top/bottom), badge padding |
| md | 16px | Roster column horizontal padding, button internal padding |
| lg | 24px | Controls block padding (above/below button group) |
| xl | 32px | Column header padding |
| 2xl | 48px | Not used in Phase 2 |
| 3xl | 64px | Not used in Phase 2 |

Exceptions:
- Color swatch: 12px wide × 12px tall (between xs and sm — matches roster row height rhythm without overpowering the name)
- Roster row min-height: 28px (ensures 20 rows fit in a narrow column without scrolling at typical desktop viewport heights ≥ 768px)
- Button min-height: 36px (touch-target baseline, desktop context)

## Color

This is a game UI with a dark, focused aesthetic appropriate for racing/sports.
Source: base.css provides a foundation; Phase 2 replaces scaffold tokens with game tokens.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#1a1a2e` | App background, column backgrounds |
| Secondary (30%) | `#16213e` | Roster panel surface, placeholder column surfaces |
| Accent (10%) | `#e94560` | Generate button, Start button (active/enabled state only) |
| Destructive | — | No destructive actions in Phase 2 |

Accent (`#e94560`) is reserved for:
- Generate button background (when enabled)
- Start button background (when enabled)
- No other elements in Phase 2

Text colors:
- Primary text: `#e0e0e0` (horse names, button text, headings)
- Muted text: `#6b7280` (placeholder column text, disabled button text)
- Condition score badge background: `#0f3460`; badge text: `#e0e0e0`

Column dividers: 1px solid `#2d3748`

Color swatch: horse's own `horse.color` hex value (inline, from Vuex state — 20 distinct colors pre-baked in Phase 1).

Disabled state color (source: CONTEXT.md D-04 — standard visual treatment):
- Button background: `#2d3748`
- Button text: `#6b7280`
- Cursor: `not-allowed`
- Opacity: 0.6 on the button element

## Interaction Contract

### HorseRow (ROST-01, ROST-02)

- Color swatch: 12px × 12px square, `border-radius: 2px`, background = `horse.color`
- Layout per row: `display: flex; align-items: center; gap: 8px; padding: 4px 16px;`
- Row order: `[swatch] [name] [badge]` — swatch left, name flex-grows, badge right-aligned
- Condition score badge: pill shape (`border-radius: 4px`), 12px semibold text, background `#0f3460`, text `#e0e0e0`, padding `2px 6px`
- No hover state required in Phase 2 (rows are display-only, not interactive)

### GameControls (SCHED-01, SCHED-04, RACE-01, RACE-04)

Controls placement: above the roster list, inside the roster column. Order: Generate button first, Start button second.

**Generate button**
- Label: "Generate Schedule"
- Enabled when: `gamePhase !== 'RACING'`
- Disabled when: `gamePhase === 'RACING'`
- Dispatches: `schedule/generateSchedule`

**Start button**
- Label: "Start Race"
- Enabled when: `gamePhase === 'SCHEDULED'`
- Disabled when: `gamePhase === 'IDLE' || gamePhase === 'RACING'`
- Dispatches: `race/startRace`

Button styling (both):
- Width: 100% of roster column padding area (fills column width minus md padding on each side)
- Height: 36px
- Border-radius: 6px
- Font: 16px semibold
- Enabled: background `#e94560`, text `#ffffff`, cursor `pointer`
- Disabled: background `#2d3748`, text `#6b7280`, cursor `not-allowed`, opacity 0.6
- Spacing between buttons: 8px (sm)

No tooltip on disabled state (CONTEXT.md D-04 — standard visual treatment only).

### Placeholder Columns (center + right)

- Background: secondary color `#16213e`
- Centered text (both axes): muted text `#6b7280`, 14px regular
- Center column placeholder text: "Race Track" (single label, no further detail)
- Right column placeholder text: "Results" (single label, no further detail)
- No border, no inner padding required — label only
